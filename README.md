# Mackerel

Mackerel is a minimal static site generator written in typed Python 3.6+.

![Latest Version](https://img.shields.io/pypi/v/mackerel.svg)
[![Build Status](https://travis-ci.org/pkolios/mackerel.svg?branch=master)](https://travis-ci.org/pkolios/mackerel)
[![Coverage Status](https://coveralls.io/repos/pkolios/mackerel/badge.svg?branch=master)](https://coveralls.io/r/pkolios/mackerel)

## Installation

Install mackerel with pip:

```bash
pip install mackerel
````

Or by cloning the repository:

```bash
git clone https://github.com/pkolios/mackerel.git
```

And installing mackerel:

```bash
cd mackerel
pip install -e .
```

## Documentation

View the basic usage documentation at [mackerel.sh](http://mackerel.sh).

## Testing

Run the tests from the mackerel root directory:

```bash
pytest
```
